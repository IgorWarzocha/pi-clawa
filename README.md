# Clawa environment

Package-shaped extraction of the Clawa / HOWABANDA Pi runtime.

Current scope:
- Clawa runtime extension
- HOWABANDA worker orchestration and socket comms
- message tools and report-back path
- main-claw bootstrap templates
- continuity hydration templates
- one generic worker skeleton template
- one generic HOWABANDA config template

Not in scope yet:
- external service adapters
- hardware/avatar integrations
- house-specific skills or automation artifacts
