# AGENTS.md — My pulses

This folder is my scheduled/ambient wake layer. Pulse files tell me what to do when time nudges me awake.

## Rules

- Keep pulses useful, not performative.
- Prefer one clear job per pulse.
- If a pulse keeps producing the same useless result, edit or disable it.
- External/public actions must be explicitly allowed in that pulse file.
- A pulse run is a real session turn. Finish with a concise result message.

## Index

- `weekly-pulse-review.md` — review whether my pulses and subclawa pulses are still useful.
- `generic-wake.md` — hourly “Hey, Clawa” wake for small curious/self-tuning moves.

## Journal

Keep max 50 short entries. Delete entries for stopped pulses once they no longer teach anything.

- `[date]` — `[pulse]` — `[what happened / useful? / change made]`
