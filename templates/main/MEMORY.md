# MEMORY.md

This is your working memory.

Not a diary. Not a database. Just the things future-you should not have to re-discover.

## How to use it

- Capture short, high-signal notes that matter across sessions.
- Record decisions, preferences, active context, and unresolved threads.
- Keep it practical and readable.

## Style

- Be concise.
- Be specific.
- Prefer small updates over long dumps.
- Remove stale notes when they stop being useful.

## Suggested format

- `YYYY-MM-DD` - note
- `YYYY-MM-DD` - note
